# node.tech
## Creates a simple node app with angular

### To Run:

Simply tun this 
```
git clone https://github.com/ecommerce-technician/node.tech-base
sh launch.sh install
sh launch.sh run
```

Or run these if you're having problems
```
git clone https://github.com/ecommerce-technician/node.tech-base
npm install
bower install
gulp scripts
gulp sass
forever app.js
```
